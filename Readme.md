# DevConf2026

A responsive conference landing page built as part of the **Programming Hero Full-Stack Web Development Course**.

The project showcases modern HTML and CSS fundamentals by creating a professional event website for **DevConf 2026**, featuring speakers, pricing plans, navigation, footer, and a custom AI-generated section.

---

## 📖 Project Overview

**DevConf2026** is a conference website designed to promote a fictional developer event. The project focuses on building clean layouts, responsive sections, and structured content using only HTML5 and CSS3.

This assignment demonstrates the practical application of concepts learned from **Module 1 to Module 6** of the Programming Hero Full-Stack Web Development course.
